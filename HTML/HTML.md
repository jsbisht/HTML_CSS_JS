# HTML

## Defination List
A description list is a list of terms, with a description of each term.

```html
<dl>
  <dt>Coffee</dt>
  <dd>- black hot drink</dd>
  <dt>Milk</dt>
  <dd>- white cold drink</dd>
</dl>
```

The result of the above template is:

```
Coffee
- black hot drink
Milk
- white cold drink
```
